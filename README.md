#  Project Name: Tiles Gallery

##  Purpose

Tiles Gallery is a modern web application where users can explore, view, and manage different tiles or image-based content in a clean and organized layout.
It provides a smooth browsing experience with dynamic data rendering and user-friendly design.

---

##  Live URL

🔗 https://tiles-galley-08.vercel.app/

---

##  Key Features

*  Display tiles/images in a grid layout
*  User Authentication (Login/Register system)
*  Built with Next.js for fast performance
*  Fully Responsive (Mobile, Tablet, Desktop)
*  Clean UI with Tailwind CSS
*  Toast notifications for user actions
*  Dynamic data fetching and rendering

---

##  NPM Packages Used

* `next` – React framework for production
* `react` – UI library
* `react-hook-form` – Form management
* `react-icons` – Icon support
* `react-toastify` – Notification system
* `better-auth` – Authentication
* `animate.css` – Animation effects
